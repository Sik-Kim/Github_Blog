## GNU/LINUX

- **운영체제의 한 종류로 LINUX는 Web Services를 제공하는 Web Server를 운영하는 컴퓨터의 OS로 사용된다.**
- LINUX는 정확히 말해선 GNU/LINUX이다.

  - OS의 Kernal : LINUX

  - OS의 나머지 부분 : GNU Project

  - LINUX보다 먼저 리처드 스톨먼에 의해 GNU Project 가 개발되고 있었다. (GNU는 GNU's not Unix로 소프트웨어의 상업화를 반대하는 의미로 오픈소스로 개발했다.)

  - 이후 리누스 토발즈가 GNU Project에 의해 LINUX를 개발

- window, mac과 같이 GUI 기능도 제공

- 오픈소스로 무료이다. 오픈 소스로 풀리며 수많은 개발자들을 통해 급격한 발전을 해왔다. (UNIX, Window, Mac OS는 유료) (오픈소스의 모체급..)
